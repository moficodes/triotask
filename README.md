# TrioTask

TrioTask is a minimalist, high-impact task management application designed to keep you focused on what truly matters. By limiting the "noise" and focusing on priority-driven task management, TrioTask helps you achieve a state of flow and maintain momentum throughout your day.

## Deployment
The app is deployed on Google Cloud Run:
[https://triotask-598464211339.us-west2.run.app](https://triotask-598464211339.us-west2.run.app)

## Why TrioTask for a Productivity Hackathon?

In a world of over-engineered productivity tools, TrioTask takes a stand for simplicity and effectiveness. It's a perfect hackathon project because:

1.  **Focus over Features**: It implements the "Power of Three" — encouraging users to focus on their top priorities rather than a never-ending list.
2.  **Frictionless UX**: With global keyboard shortcuts (press `?` for help, `n` for new task), adding and managing tasks is nearly instantaneous.
3.  **Visual Feedback**: Subtle animations and confetti celebrations provide positive reinforcement, making productivity feel rewarding.
4.  **Priority-First**: High, medium, and low priority levels are visually distinct, ensuring you're always working on the most impactful tasks.

## Tech Stack
- **React 19**: Leveraging the latest React features for a smooth UI.
- **Vite**: For blazing-fast development and optimized builds.
- **Tailwind CSS v4**: Modern styling with a utility-first approach.
- **Motion**: Fluid animations that enhance the user experience.
- **Cloud Run**: Scalable, serverless deployment.

## Keyboard Shortcuts
- `?`: Toggle shortcuts help
- `n`: Focus new task input
- `Esc`: Blur input / Close help / Cancel edit
- `Shift + C`: Clear completed tasks
- `p`: Toggle priority sorting
